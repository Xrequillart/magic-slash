Split view

Run two agents side by side in a split terminal view. Drag and drop agents between the left and right panes to reorganize on the fly. Each pane scrolls and focuses independently — perfect for coding in one while reviewing a PR in the other, or comparing two implementations without ever switching tabs.

Live agent tracking

Every agent displays its real-time status — idle, working, waiting for input, or completed. Agents are grouped by workflow stage (backlog, in progress, in review, done) so you always know where things stand. When an agent finishes a task or needs your attention, you get a native macOS notification — no need to keep checking.

Keyboard-first

Every action in the app is mapped to a keyboard shortcut. Navigate agents, toggle views, switch panes, create and manage tasks — all without reaching for the mouse. Designed for developers who live in the terminal and expect the same speed everywhere.

Context panel

A resizable sidebar dedicated to the active agent, showing everything you need: the linked Jira ticket with its title, description, and status badge; the current Git branch with uncommitted file changes and a visual additions/deletions gauge; unpushed commits with short hashes and relative dates; and the associated PR with its review status. Everything updates in real time as the agent works.

Script runner

Run your package.json scripts directly from the context panel — dev, build, test, lint — without typing a single command. No more switching to a separate terminal window just to kick off a build or run your test suite.

Test results from Vitest, Jest, and Mocha are automatically parsed and surfaced as toast notifications with pass/fail counts. If a test fails, the agent is flagged immediately so you can jump in and fix it.

Scripts run in the background while your agents keep working. You get real-time output streaming, exit code tracking, and a full log you can scroll through at any time. One click to start, one click to stop — your entire toolchain lives right next to your code.

Auto-updates

The app checks for updates automatically in the background. When a new version is available, it downloads silently with a progress indicator — no interruptions. After restart, a "What's New" modal walks you through the latest changes with full release notes. Always up to date, zero effort.
